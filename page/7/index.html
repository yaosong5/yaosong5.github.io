提交

拉取

git pull <远程主机名> <远程分支名>:<本地分支名>
如拉取远程的 master 分支到本地 wy 分支：
git pull origin master:wy

git 放弃本地修改 强制更新

git fetch 只是下载远程的库的内容，不做任何的合并 git reset 把HEAD指向刚刚下载的最新的版本
